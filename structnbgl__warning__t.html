Detailed Description

The necessary parameters to build a warning page preceding a review. One can either use predefinedSet when the warnings are already supported in nbgl_warningType_t list, or use introDetails or reviewDetails to configure manually the warning pages.

Note
it's also used to build the modal pages displayed when touching the top-right button in the review's first page
