Detailed Description

Hash functions.

A hash function maps data of arbitrary size to a bit array of a fixed size, called the message digest. Various hash functions are available:

  • BLAKE2B
  • KECCAK (Pre SHA3)
  • RIPEMD-160
  • SHAKE-128
  • SHAKE-256
  • SHA224
  • SHA256
  • SHA3
  • SHA384
  • SHA3_256
  • SHA3_512
  • SHA512
